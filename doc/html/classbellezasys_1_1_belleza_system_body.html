Detailed Description

classe Body responsavel por guardar os dados do BellezaSystem.

e dona (por ponteiro, alocados no heap) de todo Usuario/Servico/ Profissional/Agendamento cadastrado no salao. os quatro cadastros ficam em vetores (nao em mapas) pra permitir expor iteradores diretos sobre eles, igual o ModelBody faz com systems/flows.
