Detailed Description

orquestra o modelo de dominio do BellezaSys.

guarda todo Usuario/Servico/Profissional/Agendamento cadastrado no salao e garante as regras de agendamento (expediente, conflito de horario, papel do cliente). as instancias sao controladas por um registro global, do mesmo jeito que Model faz com System/Flow nos exemplos da disciplina. use a fabrica estatica createModel() para criar e registrar um sistema, e deleteModel() para desregistrar e destruir.

nenhum metodo retorna uma colecao inteira por valor: toda consulta que devolve varios itens expoe um par Begin()/End() de iterador, igual ao systemsBegin()/systemsEnd() do Model.
