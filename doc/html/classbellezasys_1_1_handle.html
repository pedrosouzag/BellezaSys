Detailed Description

template<class T>
class bellezasys::Handle< T >

classe Handle<T>: implementa o padrao "bridge" (handle/body). representa a interface publica de uma entidade, delegando os dados reais para um Body do tipo T (pImpl_). copiar um Handle nao duplica o Body, so incrementa a contagem de referencias.
